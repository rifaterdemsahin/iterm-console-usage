# How Warp Makes Money

Warp is a modern, AI-powered terminal and development environment. Here's how it generates revenue:

## 1. Freemium SaaS Model

Warp offers a **free tier** for individual developers and charges for advanced features via paid plans.

### Free Tier
- Core terminal features
- Basic AI assistance (limited usage)
- Local usage

### Paid Plans
- **Warp Pro / Teams**: Unlocks higher AI usage limits, team collaboration features, shared Warp Drive (workflows, notebooks, environment variables), and priority support.
- **Enterprise**: Custom pricing for large organizations, with SSO, audit logs, advanced security controls, and dedicated support.

## 2. AI Features (Usage-Based)

Warp integrates AI deeply into the terminal — for command suggestions, error explanations, and the Oz AI agent. Heavy AI usage beyond the free tier quota drives upgrades to paid plans.

## 3. Warp Drive (Collaboration Layer)

Warp Drive lets teams share:
- **Workflows** (saved commands with parameters)
- **Notebooks** (runbooks and documentation)
- **Environment variables** (secrets and config)

Team and Enterprise plans unlock collaborative Warp Drive features, making it a productivity platform beyond just a terminal.

## 4. Oz Cloud Agents

Warp offers **cloud-based AI agents** (Oz) that can run long-horizon development tasks autonomously in configured cloud environments. This is a premium feature targeted at engineering teams and enterprises.

## 5. Enterprise Contracts

For larger organizations, Warp offers:
- Volume licensing
- Custom SLAs
- On-premise or private cloud deployment options
- Compliance and security features (SOC 2, SSO/SAML, audit logging)

## Summary

| Revenue Stream        | Target Audience          |
|-----------------------|--------------------------|
| Pro subscriptions     | Individual power users   |
| Team plans            | Small to mid-size teams  |
| Enterprise contracts  | Large organizations      |
| AI usage tiers        | All users                |
| Cloud agent compute   | Teams using Oz agents    |

Warp's core strategy is to land developers with a best-in-class free terminal, then expand into teams and enterprises through collaboration, AI, and automation features.
